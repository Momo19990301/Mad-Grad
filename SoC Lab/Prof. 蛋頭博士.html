# 第一作者 -

研究生發表您傳出的論文時，必須從手牌中挑選同色論文發表在您面前。

# 您是對的 -

您的所有動作牌都可以造成雙倍效果，或是觸發抽牌。

# 您懂不懂色彩學？ -

給一位研究生三篇您已發表論文作為手牌，請他多讀點書，並且將他一篇已發表論文變為垃圾論文。

# 斷簡殘篇 -

一位研究生申請口試並且發表過垃圾論文，可以要求他再發表一篇研究相關論文才能口試，並且撤銷所有他的垃圾論文。

# 念能力－轉貼生 -

您可以用三張手牌將非傳遞的論文轉貼給另一位研究生發表。
